prometheusдёӯcadvisorзӣёе…іжҢҮж Ү
еҸ‘иЎЁдәҺ2019-10-23|жӣҙж–°дәҺ2020-05-14|kubernetesprometheus
|йҳ…иҜ»йҮҸ:

prometheusдёӯcadvisorзӣёе…іжҢҮж Ү

###гҖҖе®№еҷЁзӣ‘жҺ§зҡ„еҶ…еӯҳзӣёе…іжҢҮж Ү

еҗҚз§° зұ»еһӢ еҚ•дҪҚ иҜҙжҳҺ
container_memory_rss gauge еӯ—иҠӮж•° bytes RSSеҶ…еӯҳпјҢеҚіеёёй©»еҶ…еӯҳйӣҶпјҲResident Set SizeпјүпјҢжҳҜеҲҶй…Қз»ҷиҝӣзЁӢдҪҝз”Ёе®һйҷ…зү©зҗҶеҶ…еӯҳпјҢиҖҢдёҚжҳҜзЈҒзӣҳдёҠзј“еӯҳзҡ„иҷҡжӢҹеҶ…еӯҳгҖӮRSSеҶ…еӯҳеҢ…жӢ¬жүҖжңүеҲҶй…Қзҡ„ж ҲеҶ…еӯҳе’Ңе ҶеҶ…еӯҳпјҢд»ҘеҸҠеҠ иҪҪеҲ°зү©зҗҶеҶ…еӯҳдёӯзҡ„е…ұдә«еә“еҚ з”Ёзҡ„еҶ…еӯҳз©әй—ҙпјҢдҪҶдёҚеҢ…жӢ¬иҝӣе…ҘдәӨжҚўеҲҶеҢәзҡ„еҶ…еӯҳгҖӮ
container_memory_usage_bytes gauge еӯ—иҠӮж•° bytes еҪ“еүҚдҪҝз”Ёзҡ„еҶ…еӯҳйҮҸпјҢеҢ…жӢ¬жүҖжңүдҪҝз”Ёзҡ„еҶ…еӯҳпјҢдёҚз®ЎжңүжІЎжңүиў«и®ҝй—®гҖӮ
container_memory_max_usage_bytes gauge еӯ—иҠӮж•° bytes жңҖеӨ§еҶ…еӯҳдҪҝз”ЁйҮҸзҡ„и®°еҪ•гҖӮ
container_memory_cache gauge еӯ—иҠӮж•° bytes й«ҳйҖҹзј“еӯҳпјҲcacheпјүзҡ„дҪҝз”ЁйҮҸгҖӮcacheжҳҜдҪҚдәҺCPUдёҺдё»еҶ…еӯҳй—ҙзҡ„дёҖз§Қе®№йҮҸиҫғе°ҸдҪҶйҖҹеәҰеҫҲй«ҳзҡ„еӯҳеӮЁеҷЁпјҢжҳҜдёәдәҶжҸҗй«ҳcpuе’ҢеҶ…еӯҳд№Ӣй—ҙзҡ„ж•°жҚ®дәӨжҚўйҖҹеәҰиҖҢи®ҫи®Ўзҡ„гҖӮ
container_memory_swap gauge еӯ—иҠӮж•° bytes иҷҡжӢҹеҶ…еӯҳдҪҝз”ЁйҮҸгҖӮиҷҡжӢҹеҶ…еӯҳпјҲswapпјүжҢҮзҡ„жҳҜз”ЁзЈҒзӣҳжқҘжЁЎжӢҹеҶ…еӯҳдҪҝз”ЁгҖӮеҪ“зү©зҗҶеҶ…еӯҳеҝ«иҰҒдҪҝз”Ёе®ҢжҲ–иҖ…иҫҫеҲ°дёҖе®ҡжҜ”дҫӢпјҢе°ұеҸҜд»ҘжҠҠйғЁеҲҶдёҚз”Ёзҡ„еҶ…еӯҳж•°жҚ®дәӨжҚўеҲ°зЎ¬зӣҳдҝқеӯҳпјҢйңҖиҰҒдҪҝз”Ёж—¶еҶҚи°ғе…Ҙзү©зҗҶеҶ…еӯҳ
container_memory_working_set_bytes gauge еӯ—иҠӮж•° bytes еҪ“еүҚеҶ…еӯҳе·ҘдҪңйӣҶпјҲworking setпјүдҪҝз”ЁйҮҸгҖӮ
container_memory_failcnt counter ж¬Ў з”іиҜ·еҶ…еӯҳеӨұиҙҘж¬Ўж•°и®Ўж•°
container_memory_failures_total counter ж¬Ў зҙҜи®Ўзҡ„еҶ…еӯҳз”іиҜ·й”ҷиҜҜж¬Ўж•°

еҶ…еӯҳзӣёе…іжҢҮж ҮеӨ§е°Ҹ

container_memory_max_usage_bytes > container_memory_usage_bytes >= container_memory_working_set_bytes > container_memory_rss

е®№еҷЁзӣ‘жҺ§зҡ„CPUзӣёе…іжҢҮж Ү

еҗҚз§° зұ»еһӢ еҚ•дҪҚ иҜҙжҳҺ
container_cpu_usage_seconds_total counter з§’ж•° иҜҘе®№еҷЁжңҚеҠЎй’ҲеҜ№жҜҸдёӘCPUзҙҜи®Ўж¶ҲиҖ—зҡ„CPUж—¶й—ҙгҖӮеҰӮжһңжңүеӨҡдёӘCPUпјҢеҲҷжҖ»зҡ„CPUж—¶й—ҙйңҖиҰҒжҠҠеҗ„дёӘCPUиҖ—иҙ№зҡ„ж—¶й—ҙзӣёеҠ 
container_cpu_user_seconds_total counter з§’ж•° иҜҘе®№еҷЁжңҚеҠЎзҙҜи®Ўж¶ҲиҖ—зҡ„з”ЁжҲ·пјҲuserпјүCPUж—¶й—ҙ
container_cpu_system_seconds_total counter з§’ж•° иҜҘе®№еҷЁжңҚеҠЎзҙҜи®Ўж¶ҲиҖ—зҡ„зі»з»ҹпјҲsystemпјүCPUж—¶й—ҙ
container_cpu_cfs_throttled_seconds_total counter з§’ж•° cfs жҳҜе®Ңе…Ёе…¬е№іи°ғеәҰеҷЁ(Completely Fair Scheduler)зҡ„зј©еҶҷпјҢжҳҜLinuxзҡ„дёҖз§ҚжҺ§еҲ¶CPUиө„жәҗеҚ з”Ёзҡ„жңәеҲ¶пјҢеҸҜд»ҘжҢүжҢҮе®ҡжҜ”дҫӢеҲҶй…Қи°ғеәҰCPU зҡ„дҪҝз”Ёж—¶й—ҙгҖӮиҝҷдёӘжҢҮж ҮжҢҮзҡ„жҳҜиҜҘе®№еҷЁжңҚеҠЎиў«йҷҗеҲ¶дҪҝз”Ёзҡ„CPUж—¶й—ҙ
container_cpu_cfs_throttled_periods_total counter дёӘж•° ж–ҮжЎЈжіЁйҮҠжҳҜпјҡвҖңNumber of throttled period intervals.вҖқпјҢи§ЈйҮҠдёәиў«йҷҗеҲ¶/иҠӮжөҒзҡ„CPUж—¶й—ҙе‘Ёжңҹж•°гҖӮ
container_cpu_cfs_periods_total counter дёӘж•° ж–ҮжЎЈжіЁйҮҠжҳҜпјҡвҖңNumber of elapsed enforcement period intervalsгҖӮвҖқпјҢеә”иҜҘи§ЈйҮҠдёәе·Із»Ҹжү§иЎҢзҡ„CPUж—¶й—ҙе‘Ёжңҹж•°гҖӮ
container_cpu_load_average_10s gauge ж–ҮжЎЈжіЁйҮҠжҳҜпјҡвҖңValue of container cpu load average over the last 10 seconds.вҖқеә”иҜҘи§ЈйҮҠдёәиҝҮеҺ»10з§’еҶ…зҡ„CPUиҙҹиҪҪзҡ„е№іеқҮеҖјгҖӮ
